# 葛飾の波 — 混沌の縁

### Katsushika Wave: Edge of Chaos

---

三部作の最後に、数学を持ち込んだ。

「創発」は問いだった——秩序は命令なしに生まれるか。
「生々流転」は答えの先にある問いだった——秩序に意味はあるか。
「混沌の縁」は、その問いすら呑み込む。

秩序と混沌の境界。その薄い膜の上で、すべての複雑さが生まれる。生命も、意識も、美も。

---

## 三つの数学

### ローレンツ・アトラクタ

1963年、気象学者エドワード・ローレンツは天気予報の数値計算中に、初期値のわずかな差が全く異なる結果を生むことを発見した。決定論的な方程式が、予測不可能な振る舞いを生む。

この作品では、ローレンツ方程式が粒子の力場を生成する。アトラクタは三次元空間を永遠に彷徨うが、同じ場所を二度は通らない。粒子たちはこの見えない蝶の羽の周りを漂う。

```
dx/dt = σ(y − x)
dy/dt = x(ρ − z) − y
dz/dt = xy − βz
```

三行の方程式。これだけで無限の複雑さが生まれる。

### ジュリア集合

複素平面上の各点に対して、`z → z² + c` という単純な反復を行う。発散する点と発散しない点の境界——それがジュリア集合であり、どこまで拡大しても同じ構造が現れるフラクタルを成す。

この作品では、ジュリア集合の境界が粒子の運命を分ける膜になる。境界の内側にいる粒子は安定し、外側にいる粒子は発散する。境界の上にいる粒子は——震える。秩序と混沌のどちらにも属さず、両方に引かれながら明滅する。

パラメータ `c` は時間とともにゆっくり漂う。フラクタルの形そのものが呼吸している。

### リアプノフ指数

系がどれだけ「カオス的」であるかを定量化する数。隣り合う二つの初期条件がどれだけ速く離れていくかの指数的増加率。

```
λ > 0  →  カオス。近い軌道が指数的に発散する。
λ = 0  →  縁。秩序と混沌の境界。
λ < 0  →  秩序。摂動は収束する。
```

右上に表示される `λ` の値を見てほしい。この数が、海の状態を語っている。

## 四つの層

前二作の層はすべて生きている。

| 層 | 由来 | この作品での変容 |
|---|---|---|
| 波（Wave） | 創発 | カオスの力場と混ざり合い、予測不可能に歪む |
| 魂（Ghost） | 生々流転 | カオス領域で死んだ粒子は螺旋を描いて消える |
| 花（Bloom） | 生々流転 | フラクタル境界上での誕生はより強く輝く |
| 蔓（Tendril） | 新規 | カオス領域で死んだ粒子が残すアトラクタの軌跡 |

そして新たに加わった層：

**フラクタルの幽霊**——混沌スライダーを上げると、背景にジュリア集合の影が浮かび上がる。それは粒子が従う法則の可視化であり、見えない力場の地図であり、運命の骨格である。

**分岐のこだま**——フラクタル境界上の粒子は、ときどき自分の「もう一つの可能性」を一瞬だけ映す。少しずれた位置に、少し薄い自分が現れる。初期値がわずかに違っていたら辿ったであろう、もう一つの軌道。

## コードの中の形而上学

```
attractorField    — 見えない蝶が生む風
fractalDensity    — 運命の膜。ここを境に軌道が分岐する
juliaEscape       — 脱出できるか。できないか。その問いの反復
divergence        — 二つの人生が、どれだけ速く離れていくか
lyapunov          — 世界の予測可能性の尺度
tendril           — 消えた者が残す、アトラクタの記憶
```

`juliaEscape` 関数を読んでほしい。`z → z² + c` を繰り返すだけだ。たったこれだけの反復が、無限の複雑さを生む。そしてこの関数が返す値——発散までの速さ——が、粒子の生と死を分ける。

パラメータ `c` は時間とともに漂う。つまり、「運命を決める法則そのもの」が変化し続けている。いま安定している粒子が、次の瞬間にはカオスの渦に呑まれるかもしれない。その逆もまた。

## 決定論と自由意志

この作品のすべての粒子は、完全に決定論的な方程式に従っている。乱数すらない（初期配置を除けば）。すべての軌道は、最初の瞬間にすでに決まっている。

それなのに、予測できない。

ローレンツが発見したのはまさにこのことだ。決定論と予測可能性は別のものである。すべてが書かれていても、誰にも読めない。

この作品を眺めている時、あなたは粒子の行く先を予感しようとするだろう。時にその予感は当たり、時に裏切られる。その体験は、私たちが人生そのものに対して持つ関係と、どこか似ていないだろうか。
